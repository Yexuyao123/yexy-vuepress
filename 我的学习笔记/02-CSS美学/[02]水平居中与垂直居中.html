# [02] 水平居中与垂直居中

2020 年 7 月 31 日

  • [02] 水平居中与垂直居中
    • 水平居中
      • 子元素是 div 块级元素
        • 子元素宽度不确定(自适应)
        • 子元素宽度确定
      • 子元素是 inline 内联元素
    • 垂直居中
      • 父元素高度无约束/不确定
      • 父元素高度确定(尽量避免此情况)
        • 需求：兼容 IE
        • 需求：只需兼容 chrome 或 手机移动端
    • 绝对居中(垂直居中+水平居中)
      • 子元素是 div 块级元素
      • 文字的绝对居中

# 水平居中

# 子元素是 div 块级元素

<!-- 场景 -->
<div id="father">
  <div id="child"></div>
</div>

# 子元素宽度不确定(自适应)

/* 方案一 */
#child {
  margin-left: 20px;
  margin-right: 20px;
}
/* 方案二 */
#father {
  padding-left: 20px;
  padding-right: 20px;
}
/* 效果：父元素的宽度不确定 */

# 垂直居中

# 父元素高度无约束/不确定

/* 不可用方案：不正交，子元素margin被父元素吃掉 */
#child {
  margin-top: 20px;
  margin-bottom: 20px;
}
/* 方案 */
#father {
  padding-top: 20px;
  padding-bottom: 20px;
}
/* 效果：*/
/* 子元素高度不确定则父元素高度不确定 */
/* 子元素高度确定则父元素高度确定 */

# 绝对居中(垂直居中+水平居中)

# 子元素是 div 块级元素

/* 方案一 */
#father {
  /* 垂直居中 */
  padding: 100px 0;
}
#child {
  /* 水平居中 */
  margin: 0 auto;

# 文字的绝对居中

<!-- 场景 -->
<div class="demo">我是文字，我希望绝对居中</div>
1
2
/* 方案 */
#demo {
  /* 当字高与行高相差不大的时候，趋近于文字在行高里垂直居中 */
  font-size: 24px;
  line-height: 24px;
  /* 文字上下给相同padding 垂直居中 */
  padding: 5px 0;
  /* 水平居中 */
  text-align: center;
}
