# CSS 网格布局（Grid Layout）

  • Grid Layout——MDN (opens new window)
  • 是一个二维系统，可以同时处理行与列
  • 父元素——网格容器
  • 子元素——网格元素

# 兼容性

  • Can I Use Grid (opens new window)
  • IE:(6-9 不可用，10-11 需要加前缀)
  • Edge:(12-15 需加前缀，16-87 可用)
  • Firefox:(2-51 不可用，52-86 可用)
  • chrome:(4-56 不可用，57-90 不可用)
  • safari:(3.1-10 不可用，10.1-14 可用)
  • ios-safari:(3.2-10.2 不可用，10.3-14.2 可用)

  • Grid Track （1 行/1 列）：相邻 2 条网格线间空间
  • Grid Cell（1 单元格）：相邻 2 条横向网格线+相邻 2 条竖向网格线间空间
  • Grid Area(方形面域)：任意 2 横+2 竖网格线围成的空间

# container 容器属性

  • display
  1. grid：生成 1 个块级网格布局（block-level）
  2. inline-grid：生成 1 个内联网格（inline-level）
  3. subgrid：嵌套网格，此功能在 Firefox 71 中提供，它是目前唯一实现 subgrid 的浏览器。(所以不用)
  • grid-template-columns 竖向网格线间距/列宽
  • grid-template-rows 横向网格线间距/行高
  • grid-template-areas
  1. name:网格区域命名
  2. .:空网格单元
  3. none:未定义的网格区域
  • grid-template 是 culumns/rows/areas 合并写法
.parent {
  display: grid;
  grid-template:
    "a a a" 40px
    "b c c" 40px
    /* 先行高后列宽 */
    "b c c" 40px / 1fr 1fr 1fr;
}

# 要点：不正交

  • 当设置 Grid 布局后，该元素的 column、float、clear、vertical-align 属性将不起作用
