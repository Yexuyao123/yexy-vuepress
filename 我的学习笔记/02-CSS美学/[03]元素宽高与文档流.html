# 字高 ≠ 行高

  • font-size默认继承父元素的font-size属性，除非你覆盖手动约束覆盖。
  • font-size字高<line-height行高
  • 内联元素的高度不可控：当字高与行高较为相近时，高度可控，差距较大则高度不可控。

# 文档流 Noemal Flow

  • 文档流：即文档内部元素的流动方向

# 内联元素从左向右流动

  1. 默认的行内标签：
  • span内联元素(布局)
  • font：规定文本的字体、字体尺寸、字体颜色
  • b：粗体文本
  • strong：粗体文本，强调
  • em：把文本定义为强调的内容。
  • u：对其使用下划线
  • i：斜体文本
  • a：链接
  • img：图片
  • input：输入
  • 其中img和input为行内块元素。
  1. 内联元素从左向右流动，若流动遇到阻碍，换行继续流动。
  2. 中文元素会出现头部文字在上一行尾部，尾部文字跳转下一行头部的现象。
  3. 英文会将被认为是一个单词的元素放置于同一行，即一整个单词跳转下一行。(word-break:break-all;在任意位置打断)

# 块级元素从上往下流动

  • div默认为块级元素(布局)
  • div 的高度由其内部文档流的高度的总和决定
  • 块级元素每一块都独占整行，下一个块级元素跳转下一行，从上向下排布。

# 脱离文档流

  • float：left;浮动元素
  • display:fixed;固定定位
  • display:absolute;绝对定位
  • 脱离文档流的元素，不计入总高度

# 相对定位不脱离文档流

  • display:relative;相对定位不脱离文档流，保证在文档中占有一席之地。
  • 给相对定位加入定位 top 和 left，会给该元素一个定位，但是原占位依然保留。

# 上下行两端对齐

  • 1 个空格=1 个空格
  • 多个空格=1 个空格
  • 希望有多个空格，使用&nbsp代替空格(no break space)
  • 1 个空格的宽度，由字体的设计师决定，并不统一
  • 英文没有上下行对齐要求，需求：解决中文上下行两端对齐
<div>
  <span>姓名</span>
  <span>联系方式</span>
</div>

  • 使用空格模拟间距(空格的宽度由字体的设计师决定，并不统一)
  • 宽度=字数最多行宽 4em 来设定元素宽度 (麻烦，且仅适用于中文，与英文不能适应)
  • text-align: justify;在对行文字时可调整是的上下文两端对齐，但对于单行文字没有效果 (仅适用于多行文字)
  • 最佳方案：
span {
  background-color: red;
  display: inline-block;
  /* 宽度>最大行宽 */
  width: 100px;
  /* 调整多行文字两端对齐 */
  text-align: justify;
  /* 设置span高度与行高一致，并将超出部分隐藏 */
  height: 20px;
  line-height: 20px;
  overflow: hidden;
}
span::after {
  /* 将span伪装成多行文字 */
  display: inline-block;
  content: "";
  width: 100%;
  /* 查看伪类 */
  /* box-sizing:border-box; */
  /* border:1px solid green; */
}

# 内联元素空格 bug

  • 在 inline 和 inline-block 元素中

  • 1 个空格=1 个空格

  • 多个空格=1 个空格

  • 空格、回车、换行、tab 键等任何你看不见的地方字符，都会被渲染成 1 个空格，并且无法消除。

  • HTML 特点：

  • 将内联元素之内的所有看不进啊的字符显示成一个空格

  • 将内联元素之外的内容删除

  • 使用内联元素的主要目的是实现横向排布布局
  • 建议使用 float+clearfix 或 display:grid/flex 代替

# 文字的绝对居中

<!-- 场景 -->
<div class="demo">我是文字，我希望绝对居中</div>
1
2
/* 方案 */
#demo {
  /* 当字高与行高相差不大的时候，趋近于文字在行高里垂直居中 */
  font-size: 24px;
  line-height: 24px;
  /* 文字上下给相同padding 垂直居中 */
  padding: 5px 0;
  /* 水平居中 */
  text-align: center;
}
