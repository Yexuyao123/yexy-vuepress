# 自适应与响应式

  • 自适应布局：通过检测视口分辨率，来判断当前访问的设备
  • 响应式布局：针对不同客户端在客户端做代码处理。
  • 自适应要开发多套界面，响应式只要开发一套界面。
  • 自适应适配是在一定范围，响应式布局是一套页面全部适应。

# CSS 引入

<!-- 通用 -->
<link rel="stylesheet" href="/style.css" />
<!-- 打印时才生效 -->
<link rel="stylesheet" href="/print.css" media="print" />
<!-- 设备宽度符合限制时生效 -->
<link
  rel="stylesheet"
  href="/mobile.css"
  media="screen and (max-width=500px)"
/>

  name="viewport"
  content="width=device-width,user-scalable:no,nintial-scale=1.0,maxinum-scale:1.0,mininum-scale:1.0"

# 手机交互的特点

  • 没有 hover 事件
  • 有 touch 事件(touchstart、touchmove、touchend、touchcancel)
  • 如何实现 swiper(两次 touchmove 的位置差判断滑动方向)
  • 没有 resize
  • 没有滚动条
/* 隐藏滚动条 */
.xxx {
  width: 100%;
  height: 100%;
  overflow: scroll;
  /*或 overflow:auto; */
}
.xxx::-webkit-scrollbar {
  display: none;
}
