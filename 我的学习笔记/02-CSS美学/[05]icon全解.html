# iconfont 网站使用别人画好的图标

  • iconfont-阿里巴巴矢量图标库官方 (opens new window)

# 什么是 iconfont

  • 图标字体字符
  • 本质上是别人设计出来的字体，通过特殊的转译码转译。就像钢筋符号要在 CAD 中使用，就要在字体库中安装对应的特殊字体。
  • &#xe6··是 html 转译码，e6XX是统一的，从e600开始往后是预留编码区
  • iconfont 图标通过 CSS 中的font-size属性修改大小，如font-size:20px;
  • iconfont 图标通过 CSS 中的color属性修改大小，如color:red

# 修改图标大小等属性

  • 别人设计好的 iconfont 源码中可能已经被设置了大小font-size:16px;，写在.iconfont的 class 中
  • 解决方案：选用级别高于 class 的 css 选择器来实现
  • 方案 1：同级 class 进行覆盖
  • 方案 2：使用 id 选择器
  • 同理，其他属性，如字体颜色也可以这样去自定义覆盖

# 方法 1：@font-face 在线字体+ 标签插入 Unicode

  • 生成在线字体代码
  • @font-face指定一个用于显示文本的自定义字体；字体能从远程服务器或者用户本地安装的字体加载。
  • @font-face如果提供了 local()函数，从本地查找指定的字体名称，且，找到一个匹配项本地字体就会被使用。否则, 字体就会使用 url()函数下载的资源。
  • 将字体编码 Unicode 插入到 html 中
<i class="iconfont">&#x33;</i>
1
@font-face {
  /* 生成在线字体代码 */
  font-family: "iconfont"; /* project id 2449583 */
  src: url("//at.alicdn.com/t/font_2449583_z85xgevk3l.eot");
  src: url("//at.alicdn.com/t/font_2449583_z85xgevk3l.eot?#iefix") format("embedded-opentype"),
    url("//at.alicdn.com/t/font_2449583_z85xgevk3l.woff2") format("woff2"),
    url("//at.alicdn.com/t/font_2449583_z85xgevk3l.woff") format("woff"), url("//at.alicdn.com/t/font_2449583_z85xgevk3l.ttf")
      format("truetype"),
    url("//at.alicdn.com/t/font_2449583_z85xgevk3l.svg#iconfont") format("svg");
}
.iconfont {
  font-family: "iconfont" !important;
  font-size: 16px;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -webkit-text-stroke-width: 0.2px;
  -moz-osx-font-smoothing: grayscale;

# 方法 3：使用 font-class 引用

  • 拷贝项目下面生成的 fontclass 代码链接//at.alicdn.com/t/font_8d5l8fzk5b87iudi.css
  • 通过浏览器打开链接可以看源码可知其内容是一个.iconfont{}字体样式和一个方法 2 中的伪类.icon-xxx::before{}插入 Unicode 图标字符，本质上是 CND 引入了 css
  • 在项目 html 中使用 link 标签引入链接<link rel="stylesheet" href="//……链接" />
  • 挑选相应图标并获取类名，应用于页面<i class="iconfont icon-xxx"></i>

# 方法 4：使用 symbol(js 语法/SVG 图标)

  • symbol 是全新的使用方式，也是未来的主流
  • 步骤一：拷贝项目下面生成的 symbol 代码链接//at.alicdn.com/t/font_2449583_z85xgevk3l.js
  • 步骤二：在项目 html 的 head 标签中使用 script 标签引入链接<script src="//……链接" ></script>
  • 上面这一步的引入 js 的本质是在 html 中插入一个看不见的(wight 和 height 都为 0)的 SVG 标签
  • 该 SVG 标签里面包含了本次加购并创建的项目中的所有图标
  • 以 symbol 标签放在 svg 标签内部作为子标签，symbol 标签的 id 对应响应的图标名字
  • 步骤三：加入 CSS 代码
.icon {
  width: 1em;
  height: 1em;
  vertical-align: -0.15em;
  fill: currentColor;
  overflow: hidden;
}

  • svg 图片的优点：
  • 可以给 svg 标签写 CSS 样式，设置 wight 和 height 等属性，可对其修改大小、颜色等
  • svg 图是矢量图，和 CAD 图一样可以自由缩放大小，不会模糊

  • js 操作比较麻烦

  • 属于 XML 的操作
