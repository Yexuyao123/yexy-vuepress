# Flex 之前用什么布局

  • normal flow文档流：内联元素从左到右，块级元素从上到下
  • float+clearfloat 布局+清除浮动
  • position 绝对定位与相对定位
  • display:inline-block;做横向布局，如导航栏
  • 负 margin：利用负 margin 来扩大宽度或者产生位移（也可以用 translate 来产生位移）
  • table 布局

# CSS3 出现 Flex 布局

# 特点

  • 块级布局侧重垂直方向
  • 行内布局侧重水平方向
  • flex 布局与方向无关

# flex 适用

  • 实现空间自动分配、自动对齐
  • 适用于简单的线性布局
  • 复杂的布局要交给 grid 布局
  • grid 布局特点：
  1. 适用于内部系统，对于用户浏览器可控
  2. 对于诸多设备、浏览器仍有不兼容性

# 基本概念

  • flex container(flex 容器)
  • item(flex 项目)
  • main axis(主轴，默认水平)
  • cross axis(侧轴/交叉轴，默认垂直)
  • start(轴的开始位置，如：cross start)
  • end(轴的结束位置，如：mian end)
  • size(单个项目占据轴的空间，如：main size)

# container 容器属性

# flex-direction 方向

  • row：默认值，按行展示，从左往右
  • row-reverse：按行展示，从右往左
  • column：按列展示，从上到下
  • column-reverse：按列展示，从下至上
  • inherit：继承
  • initial：初始值

# flex-wrap 换行

  • nowrap：默认值，不换行
  • wrap：换行
  • wrap-reverse：倒叙换行
  • 注意：
  • 横向换行：容器要有宽度约束
  • 竖向换行：容器要有竖向约束

# flex-flow 方向+换行

  • 例：flex-flow：column wrap;

# justify-content 主轴方向

  • center：居中
  • flex-start：靠近主轴 start 位置
  • flex-end：靠近主轴 end 位置
  • space-around：空间在周围（空-item-空-item-空-item-空）
  • space-between：空间在中间（item-空-item-空-item）
  • inherit
  • initial

# align-item 侧轴方向

  • align-items 属性适用于所有的 flex 容器，它是用来设置每个 flex 元素在交叉轴上的默认对齐方式
  • align-items 属性用来让每一个单行的容器居中而不是让整个容器居中
  • stretch：默认值，延展，在侧轴方向的 size 相等，高度水平对齐
  • center：居中对齐（不延展）
  • baseline：基线对齐
  • flex-start：靠近侧轴 start 位置
  • flex-end：靠近侧轴 end 位置
  • inherit
  • initial

# align-content

  • align-content 用的少——MDN (opens new window)
  • 沿着弹性盒子布局的纵轴和网格布局的主轴在内容项之间和周围分配空间
  • 该属性对单行弹性盒子模型无效。（即：带有 flex-wrap: nowrap）
  • 只适用于多行的 flex 容器，并且当交叉轴上有多余空间时 flex 容器内的flex 线对齐。
  • 总结：align-items 和 align-content 区别在于：align-content 只能用于多行

# item 子元素属性

# flex-grow 增长比例

  • 用于父元素空间过多时
  • 例：
  • 1 个父元素宽：200px
  • 有 6 个子元素
  • 其中 3 个子元素宽均为：30px
  • 剩余 110px
.child:nth-child(4) {
  flex-grow: 2;
}
.child:nth-child(5) {
  flex-grow: 1;
}
.child:nth-child(6) {
  flex-grow: 8;
}

# flex-shink 收缩比例

  • 用于父元素空间不足时
  • flex 元素仅在默认宽度之和大于容器的时候才会发生收缩，其收缩的大小是依据 flex-shrink 的值
  • flex-shink——MDN (opens new window)

# flex-basis 默认大小（一般不用）

  • 例：flex-basis:10px;

# flex(以上 3 个的合并写法)

  • 实际应用
  • 不要拿带有高宽的 div 来写 flex 属性，要设定高宽就不要写 flex 属性
  • flex 一般用来写，当某个子元素已经设定了大小，剩下的元素如何分配空间
  • 布局结构容器和内容分离，一个 div 用来写布局，就不要再写入内容,在写一层孙元素用来写内容

# align-self 自身对齐方式

.parent {
  display: flex;
  justify-content: space-around;
  /* 父元素设定好：每个子元素都往侧轴起点的方向靠 */
  align-items: flex-start;
}
.child:nth-child(3) {
  /* 第三个子元素向侧轴终点方向靠 */
  align-self: flex-end;
}

# 重点：不正交

  • 当设置 flex 布局后，子元素的 float、clear、vertical-align 属性将消失
