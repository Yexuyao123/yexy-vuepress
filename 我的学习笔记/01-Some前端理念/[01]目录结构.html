# 目录结构

2021 年 1 月 30 日

  • 目录结构
    • 1. 前端目录结构
    • 2.vuepress 目录结构

# 1. 前端目录结构

.
│
├── .git(git仓库配置)
│
├── src(source未经翻译的代码/源码)
│   ├── assets (图片，如icon)
│   │   ├── img (给图片分库，按功能命名)
│   │   │   └── xxx.img
│   │   └── img (可选的)
│   ├── axios (存放ajax请求函数)
│   │   └── XXX (按功能命名)
│   │       └── xxx.js
│   ├── components (组件)
│   │   ├── Layout
│   │   ├── Form
│   │   └── Widge(小组件)
│   ├── CSS
│   ├── init(全局初始化js)
│   ├── mixins(混入，设置一些默认值js)
│   ├── router(路由管理 path映射)
│   ├── store(状态管理)
│   ├── type(ts的类型定义)
│   ├── util(工具函数js)
│   ├── view/page(页面文件)
│   ├── App.vue(vue主路口)
│   └── main.js(js主路口)
│
├── dist(distribution待发布的代码)
├── vendors(第三方代码，如jQuery)
├── node_modules(npm第三方包)
│
│
└── package.json

# 2.vuepress 目录结构

.
├── docs
│   ├── .vuepress (可选的)
│   │   ├── components (可选的)
│   │   ├── theme (可选的)
│   │   │   └── Layout.vue
│   │   ├── public (可选的)
│   │   ├── styles (可选的)
│   │   │   ├── index.styl
│   │   │   └── palette.styl
│   │   ├── templates (可选的, 谨慎配置)
│   │   │   ├── dev.html
│   │   │   └── ssr.html
│   │   ├── config.js (可选的)
│   │   └── enhanceApp.js (可选的)
│   │
│   ├── README.md
│   ├── guide
│   │   └── README.md
│   └── config.md
│
└── package.json
