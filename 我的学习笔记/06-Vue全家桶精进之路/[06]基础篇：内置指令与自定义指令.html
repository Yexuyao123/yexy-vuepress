# Vue.js 的内置指令

# 基本指令

# v-cloak

  • cloak:斗篷、掩盖
  • 作用：解决初始化慢导致页面闪动的最佳方案。
  • v-cloak 完全渲染结束后才显示该元素
<style>
  [v-cloak]: {
    display: none;
  }
</style>

<body>
  <div id="app">
    <div>{{msg1}}</div>
    <!-- 当网速很慢的时候会先显示{{msg}}然后再渲染内容，用户体验较差 -->
    <div v-cloak>{{msg2}}</div>
    <!-- 加了v-cloak之后，会在编译结束后再来渲染其内容，避免加载时的闪动 -->
  </div>
</body>

<script>
  let app = new Vue({
    el: "#app",
    data: {
      msg1: "不使用v-cloak",
      msg2: "v-cloak的使用,通常与display:none;结合使用",
    },
  });
</script>

# 条件渲染指令

# v-if、v-else-if、v-else

  • 用法：必须跟着屁股走
  • v-if 的弊端：Vue 在渲染元素时，出于效率考虑，只会渲染有变化的元素，会尽可能地重复利用已有的元素而非重新渲染，因此会出现乌龙。
  • 解决方案：加 key 属性使元素变唯一，提供 key 值可以来决定是否复用该元素。
<template v-if="loginType === 'username'">
  <label>Username</label>
  <input placeholder="Enter your username" key="username-input" />
</template>
<template v-else>
  <label>Email</label>
  <input placeholder="Enter your email address" key="email-input" />
</template>

# v-if 与 v-show 区别

  • v-if 实时渲染，页面显示的元素就渲染到 DOM 树中，页面不显示就移除该元素。
  • v-show 只渲染 1 次，并且永远存在于 DOM 树中，也就是说存在于页面中，只是切换 CSS 的 display 属性为 none 来不显示元素。

# 数组更新与排序

# 改变数组的一系列方法，可被 Vue 检测

  • .push()添加数组最后一个元素
  • .pop()删除数组最后一个元素
  • .shift()删除数组第一个元素
  • .unshift()添加数组第一个元素
  • .splice()可以添加或删除元素，返回删除的元素 三个参数：
    1. 参数 1：表示要开始操作的位置
    2. 参数 2：表示操作的长度
    3. 参数 3：可选参数，添加的元素(可添加多个)，添加到参数 1 的位置
  • .reverse()倒序数组
  • .sort()指定方法排序
<body>
  <div id="app">
    <ul>
      <li v-for="(item,index) in arr">{{index+1}}.{{item}}</li>
    </ul>
    <!-- 改变数组的一系列方法，可被 Vue 检测 -->
    <button @click="sortArr">点我重新排序</button>
    <button @click="reverseArr">点我倒序</button>
  </div>
</body>

<script>
  let app = new Vue({
    el: "#app",
    data: {
      arr: ["book", "pen", "pencil", "pack"],
    },
    methods: {
      sortArr: function () {
        this.arr.sort(function (a, b) {
          return a.length - b.length;
        });
      },
      reverseArr: function () {
        this.arr.reverse();
      },
    },
  });
</script>

# 数组的变动 解决方法

  1. .set(参数 1,参数 2，参数 3) 改变指定项
  • 参数 1：要改变的数组
  • 参数 2：数组要改变的 index(元素下标)
  • 参数 3：指定 index 要赋的值
// 全局变量Vue
Vue.set(app.arr, 0, "car");
1
2
  1. .splice()改变数组的长度

.splice()可以添加或删除元素，返回删除的元素,该方法接收三个参数：

  • 参数 1：表示要开始操作的位置
  • 参数 2：表示操作的长度
  • 参数 3：可选参数，添加的元素(可添加多个)，添加到参数 1 的位置
// 全局变量Vue
Vue.splice(0);

// 没有给参数2：表示删除全部，剩下的数组长度就变为0了

# 数组过滤 filter

.filter(指定过滤函数)

<body>
  <div id="app">
    <!-- 过滤：返回字符串中含有oo的项,使用计算属性 -->
    <div>含有OO的项：{{matchOO}}</div>
  </div>
</body>

<script>
  let app = new Vue({
    el: "#app",
    data: {
      arr: ["book", "pen", "pencil", "pack"],
    },
    methods: {
      getOO: function (item) {
        // .match()在字符串内检索指定的值
        console.log(item);
        return item.match("oo");
      },
    },
    computed: {
      matchOO: function () {
        return this.arr.filter(this.getOO);
      },
    },
  });
</script>

# 总结

  • v-if 结果为 true 的时候才会渲染
  • v-else-if 紧跟 v-if 才有用，v-if 为 false 的时候渲染
  • v-else 紧跟 v-if 或 v-else-if 才有用，v-if 和 v-else-if 均为 false 的时候渲染
  • v-show 结果为 false 的时候为添加行内样式"display:none;"
  • v-text 替换文本
  • v-html 渲染 html 格式的文本
  • v-on 触发事件
  • v-bind 动态修改元素属性
  • v-once 只触发一次事件
  • v-cloak 完全渲染结束后才显示该元素

# 如何自定义指令

# 自定义指令的基本用法 Vue.directive("",{})

  • 和组件类似分全局注册和局部注册
  • 区别就是把 component 换成directive

# 钩子函数（自定义指令的选项）

指令定义函数提供了几个钩子函数(可选)

  • bind：只调用一次，指令第一次绑定到元素的时候调用，用这个钩子函数可以定义一个绑定时执行一次的初始化动作。
  • inserted：被绑定元素插入到父节点时调用(父节点存在即可调用，不必存在于 document 中)。
  • update：被绑定元素所在的模板更新时调用，无论绑定值是否变化。通过比较更新前后的绑定值，可以忽略不必要的模板更新。
  • componentUpdated：被绑定元素所在模板完成一次更新周期时调用。
  • unbind：只调用一次，指令与元素解绑时调用。

# 钩子函数的参数

  • el：指令所绑定的元素，可以用来直接操作 DOM
  • binding：一个对象，包含以下属性：
    • name：指令名，不包括 v-前缀
    • value：指令的绑定值，例如：v-my-directive="1+1",其 value 的值是 2
    • oldValue：绑定指令的前一个值，仅在 update 和 componentUpdated 钩子中可用，无论值是否改变都可用
    • expression：绑定值的字符串形式，例如：v-my-directive="1+1",其 expression 的值是"1+1"
    • arg：argument,传给指令的参数，例如：v-my-directive:foo,其 arg 的值是"foo"
    • modifiers：一个包含修饰符的对象，例如 v-my-directive.foo.bar，修饰符对象 modifiers 的值是{foo:true,bar:true}
  • vnode：Vue 编译生成的虚拟节点
  • oldVnode：上一个虚拟节点，仅在 update 和 componentUpdated 钩子中可用
<body>
  <!-- 自定义指令 -->
  <div id="app">
    <div id="yxy" v-yyy.foo="lzh"></div>
  </div>
</body>
<script>
  // 自定义全局指令
  Vue.directive("yyy", {
    bind: function (el, binding, vnode) {
      // 查看el 指绑定的DOM元素
      console.log(el);
      el.innerHTML = "lalalalalalalalalala……";
      // 查看binding对象，查看其属性
      console.log(binding);
      console.log(binding.name);
      console.log(binding.expression);
      // 查看Vue编译生成的虚拟节点
      console.log(vnode);
    },
  });
  const app = new Vue({
    el: "#app",
    data: {
      lzh: "我是lzh绑定值",
    },
  });
</script>

      el.focus();
    },
  });
  const app = new Vue({
    el: "#app",
  });
</script>
