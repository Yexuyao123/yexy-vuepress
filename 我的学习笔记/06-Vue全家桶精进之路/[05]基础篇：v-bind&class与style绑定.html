# v-bind 以及 class 与 style 绑定

应用场景：DOM 元素经常会动态地绑定一些 class 类名或 style 样式。

# 了解 bind 指令

链接的 href 属性和图片的 src 属性都被动态设置了，当数据变化时，就会重新渲染。 v-bind 的作用：绑定活的属性。

<a :href="aUrl">点我</a>
<img :src="imgUrl" alt="" />

<script>
  data: {
    aUrl: "http:/baidu.com",
    imgUrl:"http:/xxx.xxx"
  }
</script>

# 对象语法

  • 给 v-bind:class 设定一个对象，可动态切换 class，值对应为 true/false
  • 当 class 的表达式过长/逻辑复杂时，可绑定计算属性 (友好且常见用法)
  • 当条件多于两个时，可使用 data/computed

# 绑定 class 对象语法 v-bind:class="{类名：布尔}"

class 绑定对象，对象的键是类名，键值是布尔值

<!-- 文本插值，插入对象 -->
<body>
  <div id="app">
    <div
      :class="{
      objStyle:isActive,
      greenBorder:hasGreenBorder
    }"
    >
      {{msg}}
    </div>
  </div>
</body>
<script>
  let app = new Vue({
    el: "#app",
    data: {
      msg: "绑定class对象语法:对象的键是类名,键值是布尔值",
      isActive: true,
      hasGreenBorder: true,
    },
    computed: {},
  });
</script>

# 绑定 class 对象语法 v-bind:class="计算属性"

计算属性返回对象，对象的键是类名，键值是布尔值

<!-- 文本插值，插入计算属性，返回对象 -->
<body>
  <div id="app">
    <div :class="className">{{msg}}</div>
  </div>
</body>
<script>
  let app = new Vue({
    el: "#app",
    data: {
      msg: "绑定class对象语法:使用计算属性 返回对象{类名：布尔}",
      isActive: true,
      hasGreenBorder: true,
    },
    computed: {
      className: function () {
        return {
          objStyle: this.isActive,
          greenBorder: this.hasGreenBorder,
        };
      },
    },
  });
</script>

# 绑定 class 数组语法 v-bind:class="[成员 1，成员 2]"

数组的成员直接对应类名

<style>
  .objStyle {
    background-color: hotpink;
    width: 200px;
    height: 200px;
    text-align: center;
  }
</style>

<!-- 文本插值，插入数组 -->
<body>
  <div id="app">
    <div :class="[normalClass]">{{msg}}</div>
  </div>
</body>
<script>
  let app = new Vue({
    el: "#app",
    data: {
      msg: "绑定class数组语法，数组中成员直接对应类名",
      normalClass: "objStyle",
    },
    computed: {},
  });
</script>

# 对象语法与数组语法混用 v-bind:class="[{类名 1：布尔}，{类名 2：布尔}，成员 3，成员 4]"

  • 数组语法多用于固定/长期绑定的 class
  • 对象语法多用于特定情况绑定的 class
<body>
  <div id="app">
    <div :class="[{greenBorder:hasGreenBorder,},normalClass]">{{msg}}</div>
  </div>
</body>

<script>
  let app = new Vue({
    el: "#app",
    data: {
      msg: "对象语法与数组语法混用",
      hasGreenBorder: true,
      normalClass: "objStyle",
    },
  });
</script>

# 绑定 style 内联样式

使用 v-bind:style 可以给元素绑定内联样式，方法与:class 类似。 也有对象语法和数组语法，看起来像是直接写在元素上的 CSS。

  • css 属性名使用驼峰命名或短横分隔命名
  • 即：字体大小尺寸，要么写 fontSize，要么写 font-size
  • 切记：Vue 中只要是(大写字母)，都会默认转化为(-小写字母)

# 对象语法

绑定内联样式对象语法：键表示 style 属性，值表示对应属性值。

<body>
  <div id="app">
    <div
      :style="{
        'width':width+'px',
        'height':height+'px',
        'background-color':bgc
      }"
    >
      {{msg}}
    </div>
  </div>
</body>
<script>
  let app = new Vue({
    el: "#app",
    data: {
      msg: "绑定style内联样式 对象语法",
      width: 200,
      height: 200,
      bgc: "yellow",
    },
    computed: {},
  });
</script>
