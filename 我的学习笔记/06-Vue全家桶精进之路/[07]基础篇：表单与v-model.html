# Vue.js 的表单和 v-model

# 基本用法

# v-model 用于 input 和 textarea

  • v-model 的双向绑定是假的,是 vue 通过特定的手法造出双向绑定的效果,曲线救国
  • 用于表单类元素上双向绑定事件
  • 可用于 input 输入框,textarea 多行文本输入框等
  • 注意:所显示的值只依赖于所绑定的数据,不再关心初始化时插入的 value
<body>
  <div id="app">
    <!-- 单行文本输入框 -->
    <input type="text" v-model="value" />
    <div>input我输入了:{{value}}</div>
    <!-- 多行文本输入框 -->
    <textarea name="" id="" cols="30" rows="10">
      我是多行文本的初始化值
    </textarea>
    <!-- 多行文本输入框v-model绑定数据后,不关心初始值是多少,只关心绑定的数据值 -->
    <textarea name="" id="" cols="30" rows="10" v-model="textareamsg">
      我是多行文本的初始化值,绑定v-model后初始化值就没有了
    </textarea>
    <div>textarea多行文本我输入了:{{textareamsg}}</div>
  </div>
</body>

<script>
  let app = new Vue({
    el: "#app",
    data: {
      value: "",
      textareamsg: "",
    },
  });
</script>

# 单选按钮 input[type="radio"]

  • 单选框只可选中,不可取消选中
  • 单个单选按钮, 直接用 v-bind 绑定一个布尔值
  • 多个单选框组合使用,就需要 v-model 来配合 value 使用,绑定选中单选框的 value 值
  1. value 值用于绑定选框和选项
  2. 使用同一个 name 将多个选框绑定
  3. 选框 v-model="selectvalue"将 数据-当前选中 value 值 相互绑定
  4. selectvalue 数据初始值为""则没有初始选中值
<body>
  <div id="app">
    <!-- 单选框 -->
    <!-- 单个单选框 初始值由v-bind=true绑定一个布尔值 -->
    <input type="radio" name="" :checked="oneradio" />
    <!-- 多个单选框 -->
    <div>
      <!-- 不指定同一个name,可以多选 -->
      <input type="radio" name="" value="泡芙" /> 泡芙
      <input type="radio" name="" value="哈十一" /> 哈十一
      <input type="radio" name="" value="狗蛋" /> 狗蛋
    </div>
    <div>
      <!-- 指定同一个name,多选框只能选定1个,v-model绑定的是你所选中的value的值 -->
      <!-- selectvalue有给定初始值的话,多选框初始选中value为该值的选框 -->
      <input type="radio" name="checks" value="泡芙" v-model="selectvalue" />
      泡芙
      <input type="radio" name="checks" value="哈十一" v-model="selectvalue" />
      哈十一
      <input type="radio" name="checks" value="狗蛋" v-model="selectvalue" />
      狗蛋
    </div>
  </div>
</body>

<script>
  let app = new Vue({
    el: "#app",
    data: {
      textareamsg: "",
      oneradio: true,
      selectvalue: "",
    },
  });
</script>

# 复选框 input[type="checkbox"]

  • 单选框只可选中,不可取消选中; 而复选框可以选中和取消选中
  • 单个复选框,直接绑定一个布尔值,可以用 v-model,也可以用 v-bind:checked
<body>
  <div id="app">
    <!-- 复选框 -->
    <!-- 单个复选框,直接绑定一个布尔值,可以用 v-model,也可以用 v-bind:checked -->
    <!-- 实测可知v-bind:checked只用于设定初始值,要与数据绑定,需要添加事件监听,通过监听函数去改变onecheckbox值,而v-model会直接与数据双向绑定 -->
    <input type="checkbox" :checked="onecheckbox" />
    <input type="checkbox" v-model="onecheckbox" />
    绑定数据onecheckbox的值{{onecheckbox}}
  </div>
</body>
<script>
  let app = new Vue({
    el: "#app",
    data: {
      onecheckbox: false,
    },
  });
</script>

  • 多个复选框,需要 v-model 来配合 value 使用,v-model 必须绑定一个数组
  • 多个复选框如果绑定的是一个字符串,则会转化为布尔值,与 checked 值相对应,由于多个复选框 v-model 值指向同一个数据,所以所有复选框会随数据值同时改变

    <input type="checkbox" value="狗蛋" v-model="selectboxvalue" /> 狗蛋
    现在选中的是{{selectboxvalue}}
  </div>
</body>
<script>
  let app = new Vue({
    el: "#app",
    data: {
      selectboxvalue: [],
    },
  });
</script>

# 下拉框 select>option

  • 默认单选下拉框,有 value 直接优先匹配一个 value 值,没有 value 就匹配一个 text 值
  • 单选下拉框使用 v-model 所绑定的 value 初始值可以为字符串或者数组
<body>
  <div id="app">
    <!-- 下拉框 -->
    <!-- 单选下拉框 select默认为单选 v-model绑定可以是字符串也可以是数组 -->
    <select name="" id="" v-model="selectoneoption">
      <option value="泡芙">泡芙</option>
      <option value="哈十一">哈十一</option>
      <option value="狗蛋">狗蛋</option>
    </select>
    现在选中的是{{selectoneoption}}
  </div>
</body>
<script>
  let app = new Vue({
    el: "#app",
    data: {
      // selectoneoption的值可以为数组或字符串
      selectoneoption: "",
    },
  });
</script>

# 总结

# 初始值

  • 单选初始值最好是给定字符串,对于单选 v-model 绑定的是静态字符串或布尔值
  • 多选初始值最好是给定数组

# 绑定值(v-model 绑定 value)

  1. 单选按钮<input type="radio" />
  • v-bind:checked用于绑定初始值,但不会绑定更新
  • v-model无法设定单选框的初始值
  • 解决方案: 使用 v-bind 动态绑定 value 值,这时用户选中时 v-model 绑定的数据会变成 value 的值
  • 单个单选按钮 v-model 要与 value 绑定时,value 必须是动态绑定
  • 多个单选按钮组件,v-model 要与 value 绑定时需给定相同的 name,value 可以通过添加属性静态绑定
<body>
  <div id="app">
    <!-- 绑定值 -->
    <!-- 使用v-bind动态绑定value值,这时用户选中时v-model绑定的数据会变成value的值 -->
    <input type="radio" v-model="inputmodel1" :value="inputvalue" />
    <div>现在v-model绑定数据值是:{{inputmodel1}}</div>
    <div>现在:value绑定的值是:{{inputvalue}}</div>
    <!-- 不使用动态绑定的value值,不会与v-model绑定 -->
    <input type="radio" v-model="inputmodel" value="111" />
    <div>现在v-model绑定数据值是:{{inputmodel2}}</div>
  </div>
</body>
<script>
  let app = new Vue({
    el: "#app",
    data: {
      inputmodel1: true,
      inputvalue: "小花",
      inputmodel2: true,
    },
  });
</script>

  1. 下拉框 select>option
  • 在 select 上绑定的 value 值,对 option 并没有影响
  • 在 option 上绑定的 value 值， v-model 与 value 绑定
<body>
  <div id="app">
    <!-- 绑定值-下拉框 -->
    <!-- 在select上绑定的value值对v-model并没有影响， -->
    <!-- typeof selectvalue值是string,而value值是对象，说明v-model未与value绑定 -->
    <select name="" id="" v-model="selectvalue" :value="{num:name}">
      <option value="泡芙">泡芙</option>
      <option value="哈十一">哈十一</option>
      <option value="狗蛋">狗蛋</option>
    </select>
    <div>现在v-model绑定数据值selectvalue是:{{selectvalue}}</div>
    <div>数据类型{{typeof selectvalue}}</div>
    <!-- 在option上绑定的value值， v-model与value绑定 -->
    <select name="" id="" v-model="selectvalue2">
      <option value="泡芙" :value="{'name':'泡芙'}">泡芙</option>
      <option value="哈十一" :value="{'name':'哈十一'}">哈十一</option>
      <option value="狗蛋" :value="{'name':'狗蛋'}">狗蛋</option>
    </select>
    <div>现在v-model绑定数据值selectvalue2是:{{selectvalue2}}</div>
    <div>数据类型{{typeof selectvalue2}}</div>
  </div>
</body>
<script>
  let app = new Vue({
    el: "#app",
    data: {
      selectvalue: "",
      selectvalue2: "",
    },
  });
</script>

# v-model 修饰符

  • v-model.lazy不实时更新
  • v-model.number将数字 str 转为数字 num
  • v-model.trim过滤输入框中字符串的首尾空格
<body>
  <div id="app">
    <!-- 修饰符 -->
    <!-- v-model.lazy -->
    <!-- v-model默认input输入时同步输入框的数据，即实时更新 -->
    <input type="text" v-model="modelStr" />
    <div>我输入了：{{modelStr}}</div>
    <!-- v-model.lazy不实时更新，用户输入完毕并按enter键后更新,或者用户鼠标点击别处(即失焦、退出输入状态)时更新 -->
    <input type="text" v-model.lazy="lazyStr" />
    <div>我输入了：{{lazyStr}}</div>

    <!-- v-model.number -->
    <!-- 注意：<input type="number"/>用户只能输入数字，但是e可以被输入，因为e是自然常数，但是输入的e数据类型是个string，如果没有对其处理极易出bug -->
    <!-- <input type="number" v-model="xxx"/>用户输入数字 得到的是string类型-->
    <input type="number" v-model="myNum" />
    <div>我输入了：{{myNum}}</div>
    <div>数据类型{{typeof myNum}}</div>
    <!-- <input type="number" v-model.number="xxx"/>用户输入数字 得到的是number类型-->
    <input type="number" v-model.number="isNum" />
    <div>我输入了：{{isNum}}</div>
    <div>数据类型{{typeof isNum}}</div>
    <!-- v-model.number也可用于<input type="text" />用户输入数字str转为数字num-->
    <input type="text" v-model.number="textNum" />
    <div>我输入了：{{textNum}}</div>
    <div>数据类型{{typeof textNum}}</div>

    <!-- v-model.trim -->
    <input type="text" v-model.trim="trimStr" />
    <div>我输入了：{{trimStr}}</div>
    <div>获取到字符串长度{{trimStr.split('').length}}</div>
  </div>
</body>
<script>
  let app = new Vue({
    el: "#app",
    data: {
      modelStr: "",
      lazyStr: "",
      myNum: "",
      isNum: "",
      textNum: "",
      trimStr: "",
    },
  });
</script>
