# 通用请求

  • 新建 helpers 文件夹

  • 新建 request.js 文档

  • 安装npm install axios

  • 配置请求信息

  • 1.import 引入插件

  • 2.生成实例.create({})

  • 3.添加请求拦截器

  • .interceptors.request.use()

  • .interceptors.response.use()

  • 4.将请求封装成函数(url,data)=>{}

  • 5.直接调用或者添加到 vue 原型链中方便在组件中调用

  • 引入 Vue 模块添加原型链Vue.prototype.$http = $axios

  • 1.请求的配置

  • 2.拦截器

  • 3.cancel 切换页面请求前取消请求
