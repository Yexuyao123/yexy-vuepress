Babel 是一个工具链，主要用于将 ECMAScript 2015+ 版本的代码转换为向后兼容的 JavaScript 语法，以便能够运行在当前和旧版本的浏览器或其他环境中。

# babel 配置及使用

# 1.babel 安装

  • babel 使用指南 (opens new window)
npm install [--save-dev可选] @babel/core @babel/cli @babel/preset-env
npm install [--save可选] @babel/polyfill
1
2

可简化：

npm i -D @babel/core @babel/cli @babel/preset-env
npm i -S @babel/polyfill
1
2

npm i -D @babel/core @babel/cli @babel/preset-env已经一键将 babel/cli、babel/core、babel/preset-env 都安装了。(以前是要分开一个个去安装的) 上述两条命令行都装完之后会在 package.json 中看到配置:

"dependencies": {
  "@babel/polyfill": "^7.12.1"
},
"devDependencies": {
  "@babel/cli": "^7.12.13",
  "@babel/core": "^7.12.13",
  "@babel/preset-env": "^7.12.13",
}

# 2.babel 配置

在项目的根目录中创建一个名为 babel.config.js 的配置文件：配置项目适用浏览器版本

const presets = [
  [
    "@babel/env",
    {
      targets: {
        edge: "17",
        firefox: "60",
        chrome: "67",
        safari: "11.1",
        ie: "6",
      },
      useBuiltIns: "usage",
      corejs: "3.6.4",
    },
  ],
];
module.exports = { presets };

# 3.babel 使用

  • 注：必须先确认 babel/cli 和 babel/core 已安装才可以使用。
  • 若未安装可以运行npm install --D @babel/core @babel/cli进行安装。
  • 命令行输入：将 src 文件夹下的 js 文件翻译并输出到 dist/js 文件夹中。
./node_modules/.bin/babel src --out-dir dist/js
1

可简化：

./node_modules/.bin/babel src -d dist/js
1

# babel 路径/全局安装与局部安装/配置环境变量

  • 从命令行可以看出 babel 必须从./node_modules/.bin/文件夹中找到。
  • 如果没有./node_modules/.bin/路径，会向上级找到根目录,根目录中没有的话，会再到全局中去找 babel(/usr/local/.bin/)，全局没有装 babel 的情况下就跑不起来。
  • 为什么在全局中就能找到/usr/local/.bin/，因为它已经被写在全局变量中的 path 里了。

可以用 npx 替换，意思是去 node_modules 文件夹中去找 bable。 再次简化：

npx babel src -d dist/js
1

# 为 babel 配置快捷键

觉得还是麻烦，还可以在 package.json 中写入快捷键，即在"scripts"键值中加入快捷 key:value。

"scripts": {
  "buildjs":"npx babel src -d dist/js",
},
1
2
3

之后可再次简化命令：

npm run buildjs
1

# 4.babel 同步监听翻译

命令行后面加--watch即可开启同步监听

npx babel js -d dist/js --watch
1

# html/img 的监听

# 监听拷贝 html

  • 运行watch -p "src/index.html" -c "cp src/index.html dist/index.html"
  • 当 src 文件夹中的 index.html 有变化，就拷贝到 dist 文件夹中。

# 监听拷贝 img

  • 同样的也可以使用该命令监听图片。
  • watch -p "src/img/**/*" -c "cp src/index.html dist/img/"
