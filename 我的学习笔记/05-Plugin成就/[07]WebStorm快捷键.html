# webpack

  • WebStorm 入门 (opens new window)
  • WebStorm 提供了对 JavaScript，TypeScript，HTML，CSS 以及 React，Angular 和 Vue.js 等框架的全面支持，无需额外的插件。
  • WebStorm 还可以帮助您使用 Node.js 开发服务器端应用程序，使用 React Native 或 Cordova 开发移动应用程序以及使用 Electron 开发桌面应用程序。建议您使用 Node.js。

# 创建文件与窗口控制

  • 创建新文件：选择要在其中创建新文件的文件夹，按Alt+Insert
  • 免打扰模式：隐藏或恢复所有工具窗口，按Ctrl+Shift+F12
  • 去项目管理：Shift+F12或F12
  • 查看最近文件：Ctrl+E

# 搜索查找

查找项 快捷键
查找任何东西 Shift+Shift
查找类 Ctrl+N
查找文件 Ctrl+Shift+N
查找命令 Ctrl+Shift+A
查找符号 Ctrl+Alt+Shift+N
当前文件中搜索文本片段 Ctrl+F
首次声明特定符号的位置 Ctrl+B 或 Ctrl+Click
  • 查找使用，显示在整个项目中使用代码元素的所有位置。Alt+F7 (被 win10 占用，可选中代码右键查找使用)

# 修复与重构

  • 显示意图动作和快速修复：修复突出显示的错误或警告，改进或优化代码构造。单击或Alt+Enter
  • 显示上下文：Alt+Enter
  • 重构：选择要重构的表达式或符号，按Ctrl+Alt+Shift+T

# 运行与调试

  • 运行任何东西：启动运行/调试配置，运行 npm/yarn，open 打开最近的项目。Ctrl+Ctrl

# 选择代码

# 根据特定的代码结构增加或减少选择的范围

  • Ctrl+W 扩展选择
  • Ctrl+Shift+W 缩小选择
  • Shift+上/下 向上/下 多选中一行
  • Shift+左/右 向左/右 多选中一个字母
  • Ctrl+Shift+左/右 向左/右 多选中一个单词

# 代码操作

  • Ctrl+C 复制当前行
  • Ctrl+Shift+上/下 选中整行代码 并将代码向上/下移动一行

# 移动光标

  • Ctrl+左/右 向左/右跳一个单词
  • Ctrl+Home/End或PgUp/PgDn 跳到文档首部/尾部

# 注释掉一行或一段代码

  • Ctrl+/ 注释当前行 或 取消注释当前行
  • Ctrl+Shift+/ 添加或删除注释块
